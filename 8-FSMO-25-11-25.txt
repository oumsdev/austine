Role FSMO 
Dans la Foret 
• Maitre Nom 
• Controleur de Schema 

Dans le Domaine 
• Maitre RID 
• Maitre Infra 
• Maitre PDC 

Maitre de Nom =>S’assurer qu’il n’y pas de domaine identique dans la fôret 
Controleur de Schema => Active Directory c’est une base de donnée, il gère la Structure (schema) d’AD 

Maitre RID => Sert à s’assurer que chaque ressources est un SID unique 

Maitre Infra => S’assurer qu’il n’y pas des doublons dans mon infra (dossier partagé ayant le meme 
nom) 

Maitre PDC =>  
Permet au vieux OS d’intégrer dans un domaine (Windows NT) 
Synchronique L‘horloge de tous les Contrôle de domaine 
Verrouille un compte utilisateur
